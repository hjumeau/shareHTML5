Introduction

L'objectif de cette soirée sur le thème HTML5, est de vous sensibiliser à ce nouveau standard du Web a travers des exemples concrets.

Et d'attiser votre curiosité pour permettre d'identifier des sujets à approfondir.

Constat

Divergence grandissante entre
  • les possibilités du langage HTML4
  • les applications web actuelles
Résultat : une belle pagaille
  • afflux de technologies côté client
  • le plein de plug-ins (flash, silverlight…)
  • divergences entre navigateurs

Support des navigateurs

    Dépend des fonctionnalités et des navigateurs
  • on peut tester les attributs et fonctions (CSS et JS)
  • les balises inconnues sont généralement ignorées
  • on peut utiliser les autres CSS pour certains navigateurs
    La bibliothèque populaire : Modernizr
  • librairie js pour tester les fonctionnalités supportées
  • inclus des polyfills
    Implante des fonctionnalités de HTML5 dans les navigateurs qui ne supportent pas nativement

Conclusion

    HTML5 est présenté comme le tueur de flash
  • De nombreuses nouveautés (HTML5, CSS3, JS API)
  • problèmes d’implémentation et de standardisation
    Tout ce que nous allons couvrir est susceptible
  • de subir des modifications
  • de ne pas fonctionner dans tous les navigateurs
  • ou de fonctionner différemment

  if (window.Notification && Notification.permission !== "granted") {
